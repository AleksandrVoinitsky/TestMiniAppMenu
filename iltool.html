Помощь

  • • Нажмите "+" для создания новой семантической метки
  • • Введите текст метки (например, "#приветствие" или "[Admin] важное")
  • • В поле JSON опишите логику обработки этой метки
  • • Используйте "Сохранить" для экспорта данных в файл
  • • Используйте "Открыть" для загрузки ранее сохраненного файла
  • • "Объединить" позволяет добавить метки из другого файла без дубликатов
  • • "Очистить всё" удаляет все метки

Типы меток

# - Метка с решеткой (синий цвет)
[...] - Квадратные скобки (голубой цвет)
[Admin] - Административная метка (красный цвет)
